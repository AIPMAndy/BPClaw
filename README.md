# LivePPT

LivePPT 是一个把传统 PPT 需求升级为“可点击切换的动态网页演示”的开源 Skill。

## What It Is

- 场景化叙事（封面 / 痛点 / 方案 / 证据 / CTA）
- 多主题风格切换（不改文案只换视觉令牌）
- 7/14 天执行计划自动生成
- 一键本地校验与 CI 校验

## Project Location

当前主内容位于：

`skills/public/dynamic-showcase-web-skill`

cd skills/public/dynamic-showcase-web-skill
